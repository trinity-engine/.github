## Introduction
> [!NOTE]
> This project is a work-in-progress.

**Trinity** is a retro, [2.5D](https://en.wikipedia.org/wiki/2.5D) [software-rendered](https://en.wikipedia.org/wiki/Software_rendering)
game engine written in [C99](https://en.wikipedia.org/wiki/C99) using the [SDL2](https://wiki.libsdl.org/SDL2/FrontPage)
library, with [Lua](https://www.lua.org/about.html) for scripting. It's inspired by engines such as [Valve's Source engine](https://en.wikipedia.org/wiki/Source_(game_engine)),
the [Doom engine](https://en.wikipedia.org/wiki/Doom_engine), the [Build engine](https://en.wikipedia.org/wiki/Build_(game_engine)),
the [Quake engine](https://en.wikipedia.org/wiki/Quake_engine), and [Roblox](https://en.wikipedia.org/wiki/Roblox).

## Platform Support
Trinity aims to support the following platforms:
- Windows
- Linux
- MacOS
